Activity: Building a Multi-Page Website with Express.js

This activity involves creating a simple multi-page website using Express.js, a popular Node.js framework. The goal is to understand how to set up an Express server, serve static files, and define routes for different web pages.

Key Concepts Covered

  • Setting up an Express.js server
  • Understand the core concepts of Express.js and its role in web development.
  • Defining routes for multiple web pages (Home, About, Contact)
  • Manage static files such as CSS, images, and JavaScript.
  • Deploy an Express.js application using GitHub and Render.
  • Using res.sendFile() to serve HTML pages
